1. **Clone the repository:**
   ```bash
   git clone https://github.com/Manav1918/python-smart-ai-chatbot.git
   cd smart-ai-chatbot
   ```

2. **Create a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

